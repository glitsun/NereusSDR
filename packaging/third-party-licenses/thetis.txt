Thetis — OpenHPSDR / Apache Labs SDR console (upstream of NereusSDR ports)

NereusSDR is a C++20/Qt6 port of Thetis (ramdor/Thetis), an open-source SDR
console for OpenHPSDR Protocol 1 and Protocol 2 radios. The NereusSDR
codebase contains substantial code translated from Thetis C# sources into
C++. The translation (expression) is NereusSDR's own copyright; the
underlying logic, algorithms, constants, and protocol handling carry
Thetis's upstream copyright chain.

Upstream repository:
  https://github.com/ramdor/Thetis

Upstream licence:
  GPL-2.0-or-later (see below)

NereusSDR elects the "or later" grant and redistributes the combined work
under GPL-3.0 via GPLv3 §5(b). The full GPLv2 text accompanies this
distribution as `GPLv2.txt`.
